## Приложение "Читательский дневник"
Приложение с использованием баз данных «Читательский дневник» позволит вести учет прочитанных и запланированных к прочтению книг, и их авторов. Предметная область: учёт и систематизация личной библиотеки и читательского опыта. Система предназначена для автоматизации выполнения следующих операций:
−	ведение каталога книг с основной информацией (название, автор, количество страниц, год издания);
−	отслеживание статуса чтения книги (“хочу прочитать”, “читаю”, “прочитана”).
−	выставление собственной оценки на прочитанную книгу;
−	организация книг по тематическим полкам (например, “Любимые книги”, “Классика”);
−	сортировка книг по полкам и статусу.
