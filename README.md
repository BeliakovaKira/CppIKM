## Приложение "Читательский дневник"
Приложение с использованием баз данных «Читательский дневник» позволит вести учет прочитанных и запланированных к прочтению книг, и их авторов. Предметная область: учёт и систематизация личной библиотеки и читательского опыта. Система предназначена для автоматизации выполнения следующих операций:
1. Ведение каталога книг с основной информацией (название, автор, количество страниц, год издания).
2. Отслеживание статуса чтения книги (“хочу прочитать”, “читаю”, “прочитана”).
3. Выставление собственной оценки на прочитанную книгу.
4. Организация книг по тематическим полкам (например, “Любимые книги”, “Классика”).
5. Сортировка книг по полкам и статусу.
## Алгоритм 
1. Создаю структуру базы данных.
2. Подключаю базу данных PostgreSQL.
3. Создаю таблицы в базе данных.
4. Создаю главное окно с таблицей для отображения книг.
5. Добавляю кнопки для основных операций.
5. Реализую функции работы с книгами.
